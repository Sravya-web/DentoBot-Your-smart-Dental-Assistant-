## ‚≠ê Project Overview  
Dentobot is built to bridge the gap between patients and dental professionals by offering simple, accurate, and friendly dental guidance. It helps users understand common dental issues, get preventive care tips, and know when they must visit a dentist.

---

## Problem Statement  
Many patients lack clear awareness about oral hygiene.  
They often:
- Search online and get confused  
- Delay dental visits  
- Forget post-treatment instructions  
- Ask repeated basic questions  
- Struggle to identify if a symptom is serious  

Dentists, on the other hand, spend a lot of time repeating the same information daily.

Dentobot solves this by acting as a **24/7 smart dental assistant**.

---

## ‚úî Key Features  
- 24/7 AI chatbot for dental queries  
- Symptom guidance for minor issues  
- Personalized oral-care reminders  
- Educational videos, infographics & FAQs  
- Clinic appointment integration (future scope)  
- Warm, friendly, non-technical interaction style

##  User Research Summary  
Dentobot uses a mixed-method research approach:

### **For Patients**
- **User Interviews** ‚Äî Understand problems, trust, expectations  
- **Online Surveys** ‚Äî Gauge openness to chatbots  
- **Usability Testing** ‚Äî Observe task flows (pain reporting, booking)

### **For Dentists / Staff**
- **Expert Interviews** ‚Äî Ensure safety & accuracy  
- **Contextual Inquiry** ‚Äî Understand clinic workflow  
- **Co-Design Workshops** ‚Äî Decide tone, language & response style  

**Research Goals:**
- Identify user pain points  
- Define communication style  
- Prioritize features  
- Understand emotional behaviour of patients  
- Ensure clinical accuracy & safety
